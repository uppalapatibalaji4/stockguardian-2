# StockGuardian

Your personal AI-powered stock agent!  
Track investments, get profit/loss, set price/profit/drop alerts via email, and chat with an AI bot.

## Features
- Add stocks with buy price, quantity, date
- Real-time price & profit/loss calculation
- Email alerts when price hits target, profit %, or drop %
- AI chat bot with predictions
- 30-day forecast + next-day prediction

## Setup
1. Enter your email and Gmail (for sending alerts)
2. Add your investments
3. Set alerts
4. Ask the AI bot anything!

**Note**: Use a **Gmail App Password** (not your regular password).  
Get it here: [https://myaccount.google.com/apppasswords](https://myaccount.google.com/apppasswords)
